# CodeAlpha_IrisFlowerClassification
# Iris Flower Classification

## Objective
To classify Iris flowers into Setosa, Versicolor, and Virginica species using Machine Learning.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset
The Iris dataset contains 150 flower samples with:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Machine Learning Model
Random Forest Classifier

## Results
- Accuracy: 100%
- Confusion Matrix generated successfully

## Conclusion
The Random Forest model successfully classified Iris flower species with high accuracy.
